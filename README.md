# image-retrieval
<img width="245" height="147" alt="image" src="https://github.com/user-attachments/assets/ad19e042-06fd-4eef-8f42-2d710a3c9daa" />

Penelitian ini menganalisis efektivitas teknik ekstraksi fitur Local Binary Pattern (LBP) dan Histogram of Oriented Gradients (HOG) dalam sistem pencarian gambar berbasis pola. Penelitian ini juga membandingkan dua metode perhitungan jarak, yaitu Jarak Euclidean dan Cosine Similarity,menilai kemiripan antara citra uji dan citra referensi.
